# Prompts prontos para o Codex

Use estes prompts dentro do Codex App ou Codex CLI.

---

## Landing page premium

```text
Siga o AGENTS.md.

Crie uma landing page premium para [produto].

Antes de implementar, proponha 3 direções visuais diferentes:
1. premium editorial
2. dark produtivo tipo Linear
3. minimal técnico tipo Vercel

Depois escolha a direção mais forte e implemente.

Não quero visual genérico de IA. Evite gradiente roxo/azul padrão, cards repetitivos, ícones aleatórios e copy clichê.
Revise todos os textos para parecerem humanos, específicos e diretos.
```

---

## Interface moderna / dashboard

```text
Siga o AGENTS.md.

Crie uma dashboard moderna para [objetivo].

A tela precisa mostrar:
- [item 1]
- [item 2]
- [item 3]
- ação principal: [ação]

Quero uma interface densa, elegante e útil. Não invente métricas falsas exageradas.
Inclua estados de loading, vazio, erro, hover e focus.
```

---

## Automação com webhook

```text
Siga o AGENTS.md.

Crie uma automação que recebe um webhook, valida os dados, executa [ação] e retorna uma resposta clara.

Inclua:
- exemplo de payload
- variáveis de ambiente
- logs
- tratamento de erro
- retries quando fizer sentido
- instruções para testar localmente
```

---

## Revisão anti-cara-de-IA

```text
Siga o AGENTS.md.

Revise esta interface/landing page e remova tudo que pareça genérico ou feito por IA.

Melhore:
- hierarquia visual
- espaçamento
- tipografia
- contraste
- copy
- microinterações
- responsividade

Evite clichês como "revolucione", "desbloqueie seu potencial", "solução inovadora", "experiência perfeita" e cards genéricos com ícones aleatórios.
```

## Melhorar projeto existente

```text
Siga o AGENTS.md.

Analise o projeto atual e proponha melhorias para deixar a experiência mais premium e menos genérica.

Depois implemente as melhorias mais importantes sem mudar o stack do projeto.
Rode build/lint/testes se existirem.
```

---

## Criar design system simples

```text
Siga o AGENTS.md.

Crie um design system simples para este projeto.
Inclua tokens de cor, tipografia, espaçamento, radius, borda, sombra e componentes base.

Não exagere. Quero algo prático para manter consistência visual nas próximas telas.
```
